# Personal Agent Business

This is your Personal Agent Business cockpit. Open this folder in Claude Code and
run `/setup`. It interviews you, gathers your operator keys one at a time, and
stands up your business step by step: your own always-on personal agent, your
selling surfaces, and on-demand client minting. Nothing technical to read.
Nothing bills without your yes.

## Runs on Mac and Windows

**macOS:** Claude Code + `jq` (`brew install jq`). bash, curl, and ssh are built in.

**Windows:** Claude Code (native installer) + Git for Windows
(https://git-scm.com/downloads/win). That one download bundles bash, curl, ssh,
and jq, and Claude Code auto-detects Git Bash to run the scripts. No WSL required.

## What you need

The operator accounts from your pre-work: Hetzner, an OpenAI (or OpenRouter)
admin key, AgentMail, Composio, Cloudflare, Vercel, and Slack.
